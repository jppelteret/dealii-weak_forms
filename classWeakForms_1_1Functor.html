Detailed Description

template<int rank_>
class WeakForms::Functor< rank_ >

A base class for other objects that represent functors.

Functors are classes that act like a function. In this context, these functors are to be able to return a value of specific rank at any point in space (be it in a bulk material, on a body's boundary, or on an interface).

Template Parameters
rank_The (tensorial) rank of the return type that this object computes.
