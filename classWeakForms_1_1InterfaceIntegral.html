Detailed Description

template<typename PredicateType = types::default_interface_integral_predicate_t>
class WeakForms::InterfaceIntegral< PredicateType >

A class that encapsulates the subset of elements that are to be considered as part of an interface integral.

This class is not typically created directly by a user, but rather would be automatically generated by a form or function integrator.
