Detailed Description

template<int dim, int spacedim = dim>
class WeakForms::Normal< dim, spacedim >

A functor that represents the normal of the cell, as evaluated at a boundary or an interface.

An example of usage:

const Normal<spacedim> normal{};
const auto N = normal.value();
Template Parameters
dimThe dimension in which the scalar is being evaluated.
spacedimThe spatial dimension in which the scalar is being evaluated.
