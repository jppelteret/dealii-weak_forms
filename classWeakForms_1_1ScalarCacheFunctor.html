Detailed Description

A functor that returns (field-dependent) scalar values upon evaluation.

This class wraps generic functions that can be evaluated at one or more numerical quadrature point at a time. The output of these functions are of scalar type, and are permitted to be a function of one or more field solutions. That the extraction of the field solution is an expensive operation, its values are cached to prevent repeated computation. Since the definition of the evaluation call is provided through a std::function, the computation pipeline can be made dependent on user data if it is passed in through the capture clause of a lambda function.
