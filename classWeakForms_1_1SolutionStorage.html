◆ dof_handlers

template<typename VectorType , typename DoFHandlerType = std::nullptr_t>
const std::vector<dofhandler_ptr_type> WeakForms::SolutionStorage< VectorType, DoFHandlerType >::dof_handlers
private

Permit each of the solution vectors to be associated with a DoFHandler that is not necessarily the one used in the main assembly loop. This allows us to use external finite element solutions as fields for source terms and the like. If the "default" option, namely associating this solution field with the primary DoFHandler, is to be made then the entry can just be null-initialised.
