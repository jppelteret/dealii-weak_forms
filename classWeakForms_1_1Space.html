Detailed Description

template<int dim, int spacedim>
class WeakForms::Space< dim, spacedim >

A base class for objects that encapsulate the notion of a (discrete) global finite element space.

Underlying this global space might be a single finite element (scalar-, vector-, or tensor-valued), or grouping of finite element spaces. The abstract space ( \( \mathcal{L}^{2}\), \( \mathcal{H}^{1}\), \( \mathcal{H}\left(\text{curl}\right) \), \( \mathcal{H}\left(\text{div}\right) \) ) or composition of spaces that this represents is somewhat arbitrary.

Template Parameters
dimThe dimension in which the scalar is being evaluated.
spacedimThe spatial dimension in which the scalar is being evaluated.
