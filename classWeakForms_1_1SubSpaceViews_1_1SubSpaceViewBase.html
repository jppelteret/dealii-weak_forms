Detailed Description

template<typename SpaceType, typename FEValuesExtractors_t>
class WeakForms::SubSpaceViews::SubSpaceViewBase< SpaceType, FEValuesExtractors_t >

A base class for objects that represent a subspace within a global finite element space.

Template Parameters
SpaceTypeA weak forms object that represents a space. This could be either a test function, trial solution, or a field solution.
FEValuesExtractors_tOne of the extractors from the deal.II FEValuesExtractors namespace.
