Detailed Description

template<int rank_>
struct WeakForms::SubSpaceExtractors::SymmetricTensor< rank_ >

An object that can be used to extract a symmetric tensor subspace from a global finite element space.

The number of tensor components is determined on the basis of the space from which the extraction is to be performed.

Template Parameters
rank_The rank of the tensor that this subspace is associated with.
