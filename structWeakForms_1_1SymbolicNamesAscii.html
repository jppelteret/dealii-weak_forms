Detailed Description

A data structure that defines the labels to be used to construct ASCII-formatted symbolic variables identifiers.

Warning
It is critical to ensure that the labels are unique. If not, then there is the possibility that one can generate conflicting symbolic expressions that will not be detected during their use.
